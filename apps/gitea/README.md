# Gitea

A lightweight, self-hosted Git service written in Go that provides repository hosting, code review, and project management. It offers a GitHub-like experience with minimal resource requirements, making it perfect for individuals and small teams who want full control over their source code.

## Key Features

- **Git hosting** — Full-featured Git repository hosting with a clean web interface, code browsing, and wiki
- **CI/CD** — Built-in Gitea Actions for continuous integration, fully compatible with GitHub Actions workflows
- **Issue tracking** — Manage work with issues, pull requests, milestones, and kanban project boards
- **Lightweight** — Written in Go with minimal dependencies, runs efficiently on low-spec hardware
