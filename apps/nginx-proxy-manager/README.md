# Nginx Proxy Manager

A Docker-based reverse proxy management tool that provides a simple web interface for managing Nginx proxy hosts, SSL certificates, and access control. It makes it easy to expose your self-hosted services to the internet with automatic HTTPS via Let's Encrypt.

## Key Features

- **SSL management** — Automatically provision and renew free SSL certificates from Let's Encrypt
- **Reverse proxy** — Configure reverse proxy hosts through an intuitive web interface without editing config files
- **Redirection** — Set up URL redirections with 301/302 status codes
- **Access control** — Protect services with IP-based access lists and HTTP basic authentication

## Default Ports

| Service | Port |
|---------|------|
| HTTP | 80 |
| Admin UI | 81 |
| HTTPS | 443 |
