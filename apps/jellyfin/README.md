# Jellyfin

A free and open-source media server that lets you organize and stream your personal media collection. Unlike competitors such as Plex or Emby, Jellyfin has no premium tiers or tracking — every feature is completely free and the project is fully community-driven.

## Key Features

- **Media streaming** — Stream your movies, TV shows, and music library from anywhere with transcoding support
- **Live TV** — Watch and record live TV with DVR support and electronic program guide (EPG)
- **Multi-client** — Native apps available for iOS, Android, Roku, Fire TV, and web browsers
- **Completely free** — No premium features locked behind a paywall, fully open source under GPL
