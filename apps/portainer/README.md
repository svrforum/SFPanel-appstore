# Portainer

A container management platform that provides a web-based UI for managing Docker environments. It simplifies Docker operations by offering an intuitive dashboard for deploying, monitoring, and troubleshooting containers without needing to use the command line.

## Key Features

- **Docker management** — Manage containers, images, volumes, and networks through a visual interface
- **Dashboard** — Get a clear overview of your Docker environment including resource usage and container status
- **Multi-user** — Set up role-based access control to manage team permissions
- **Stack deploy** — Deploy and manage Docker Compose stacks directly from the web UI
