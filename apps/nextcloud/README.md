# Nextcloud

A self-hosted productivity platform that provides file storage, syncing, and collaboration tools. It serves as a privacy-respecting alternative to Google Drive and Dropbox, with an extensive app ecosystem that adds calendars, contacts, office editing, and much more.

## Key Features

- **File sync** — Sync files seamlessly across desktop, mobile, and web with version history
- **Office suite** — Edit documents, spreadsheets, and presentations directly in the browser with Collabora or OnlyOffice
- **Calendar/Contacts** — Built-in CalDAV and CardDAV support for syncing calendars and contacts
- **App ecosystem** — Extend functionality with hundreds of community-built apps and integrations
