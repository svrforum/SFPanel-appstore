# Uptime Kuma

A self-hosted monitoring tool that keeps track of your websites, APIs, and services. It provides instant notifications when downtime is detected, supports multiple monitoring protocols, and offers beautiful status pages you can share with your team or users.

## Key Features

- **Real-time alerts** — Get notified instantly via Telegram, Discord, Slack, Email, and 90+ other notification services
- **Status pages** — Create public or private status pages with custom domain support
- **Multi-protocol monitoring** — Monitor HTTP/HTTPS, TCP, DNS, PING, GRPC, and more
- **Mobile-friendly** — Fully responsive dashboard that works seamlessly on any device
