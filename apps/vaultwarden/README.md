# Vaultwarden

An alternative implementation of the Bitwarden server API written in Rust, designed for self-hosted deployments. It is fully compatible with all official Bitwarden clients while using significantly fewer resources than the official server, making it ideal for personal and small team use.

## Key Features

- **Bitwarden compatible** — Works with all official Bitwarden apps and browser extensions
- **Password sharing** — Securely share passwords and sensitive data between users via organizations
- **2FA support** — Protect your vault with TOTP, FIDO2/WebAuthn two-factor authentication
- **Lightweight** — Uses minimal RAM and CPU, runs comfortably on low-spec servers and Raspberry Pi

## Default Ports

| Service | Port |
|---------|------|
| Web UI | 8080 |
